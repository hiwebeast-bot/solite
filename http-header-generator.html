Header Configuration

Content Security Policy (CSP)

Define which resources are allowed to load to prevent XSS attacks.

HTTP Strict Transport Security (HSTS)

Enforce secure (HTTPS) connections to the server.

Cross-Origin-Opener-Policy (COOP)

Prevents other domains from opening your site in a popup and gaining access to it.

Cross-Origin-Embedder-Policy (COEP)

Prevents a document from loading any cross-origin resources that don't explicitly grant permission.

Cross-Origin-Resource-Policy (CORP)

Controls which cross-origin sites are allowed to include your resources.
