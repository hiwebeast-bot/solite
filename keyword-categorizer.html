The Solite Keyword Categorizer is a fast, manual, drag-and-drop workspace for organizing large keyword lists into custom categories. No AI, no automation ‚Äî pure human-controlled sorting with a frictionless UI and 100% local processing. Export fully organized data in JSON or CSV formats without uploading anything to a server.

Core Features

  • Multi-Keyword Import: Paste hundreds of keywords at once (one per line).
  • Uncategorized Bucket: All new or leftover keywords stay visible until sorted.
  • Custom Categories: Create unlimited groups instantly.
  • Drag-and-Drop Sorting: Move keywords between categories with intuitive gestures.
  • Live Keyword Count: Each category dynamically shows total items.
  • One-Click JSON / CSV Export: Clean, structured, ready for SEO tools or spreadsheets.
  • Client-Side Privacy: No uploads, tracking, or cloud processing.

  • SEO Experts: Build topical clusters for content planning.
  • Content Strategists: Organize keyword research before writing.
  • PPC Specialists: Group keywords into tightly themed ad sets.
  • Marketers: Clean messy keyword dumps from tools like Ahrefs/SEMrush.

How to Use

  1. Paste keywords in the text area (one per line).
  2. Click Load Keywords to populate the uncategorized section.
  3. Add one or multiple categories from the right panel.
  4. Drag any keyword into the desired category.
  5. Rearrange or recategorize anytime by dragging again.
  6. Export your entire set as JSON or CSV.

What Your Exports Contain

  • Uncategorized List: Keywords not sorted into any category.
  • Category Buckets: Each category with its full keyword list.
